※このスクリプトを利用するにはYouTube Data API v3が必要です※

==========手順１==========
①メモ帳やエディターでスクリプト(batファイル)を開く
②ダウンロードしたフォルダのパスを指定
③保存をしてスクリプトを実行

=====コマンドプロンプト=====
@echo off
cd /d "C:\ここに指定"
call .venv\Scripts\activate
python YT-Channel_ID.py
timeout /t 3 >nul

手順１はこれで完了。

==========手順2==========
①メモ帳やエディターでYT-Channel_ID.pyを開く
②コードの中の　「API_KEY = 'YOU_API_KEY'」を自分のAPI KEYに書き換える
③保存する

手順２はこれで完了。

========================
終わったらスクリプトを実行して下さい。
